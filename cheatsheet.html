Web Development Cheatsheet

HTML

Elements and Structure

Name Description
HTML Tag The syntax for a single HTML tag is an opening bracket < folowed by the element name and a closing bracket >.
Element Content The content of an HTML element is the information between the opening and closing tags of an element.
Ex: <p>This is content</p>
HTML Element An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag.
Head Element <head> Contains general information about an HTML page that isn't displayed on the page itself.
Body Element <body> Represents the content of an HTML document that gets rendered on web browsers.
Headings <<h1> to <h6> Heading elements are ordered from the highest level <h1>, to the lowest level <h6>.
Paragraph <p> The paragraph element contains and displays a block of text.
Ordered Lists <ol>, Unordered Lists <ul>), and List Items <li> Creates a list of items in order by number, or unordered with bullet points.
Image Element <img> Embeds images in documents. <img> is an empty element (no closing tag) and the src attribute contains the image URL.
Ex: <img src="image.png">

Attributes
<input> name Attrbute In order for a form to send data, it needs to be able to put it into key-value pairs. This is achieved by setting the name attribute of the input element. The name will become the key and the value of the input will become the value the form submits corresponding to the key.
<required> Attribute Specifies that the field must include a value.
min and max Attributes Specifies the minimum or maximum value that can be entered into the field.
minlength and maxlength Attributes Makes sure that the input text has reached the minimum or is within the maximum length.

Syntax and Selectors

Name Description
<link> Element Used to lnk HTML documents to external resources like CSS files. It commonly uses href, rel, and type attributes.
Type Selectors Used to match all elements of a given type or tag name.
Class Selectors Used to match all elements based on the contents of their class attribute. These are selected by prepending the attribute with a '.'.
ID Selectors Used to match elements based on the contents of their id attrbute. These values should unique in the entire DOM. These are selected by prepending the attribute with a '#'.
