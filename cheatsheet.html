Web Development Cheatsheet

HTML

Elements and Structure

Name Description
HTML Tag The syntax for a single HTML tag is an opening bracket < folowed by the element name and a closing bracket >.
Element Content The content of an HTML element is the information between the opening and closing tags of an element.
Ex: <p>This is content</p>
HTML Element An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag.
Head Element <head> Contains general information about an HTML page that isn't displayed on the page itself.
Body Element <body> Represents the content of an HTML document that gets rendered on web browsers.
Headings <<h1> to <h6> Heading elements are ordered from the highest level <h1>, to the lowest level <h6>.
Paragraph <p> The paragraph element contains and displays a block of text.
Lists (<ol> and <ul>) and List Items <li> Creates a list of items in order (<ol> by number) or unordered (<ul> with bullet points).

CSS

Colors

Name Description Example
color The color property is used to change the foreground text color of an element.
color: red;

Hello, world!
background-color The background-color property is used to change the background color of an element.
background-color: red;

Hello, world!
Color name keywords Keywords are defined values that are used to set the color property.
color: aqua;

Hello, world!
rgb() RGB values represent how red, green, or blue a color is. Each value ranges from 0 to 255.
color: rgb(166, 108, 166);

Hello, world!
hsl() HSL values represent the hue, saturation, and lightness of a color. The hue value ranges from 0 to 360, while the others are represented as percentages.
color: hsl(29, 100%, 57%);

Hello, world!
Hexadecimal Values Hexadecimal digits can represent 16 different values using 0-9 and a-f. Hex colors are made up of 6 characters, each group of 2 characters represent a value between 0 to 255 for red, green, and blue. If all characters are repeated, hex colors can be abbreviated to 3 values.
color: #597D35;

Hello, world!
Alpha Values Alpha values determine how transparent a color will be. It can be set for RGB and HSL colors by using rgba() and hsla() respectively. This value can range from 0.0 (transparent) to 1.0 (opaque).
color: rgba(166, 108, 166, 0.35);
